# Internal

This directory contains ASCP repository workflow and planning files. It is **not** part of the public documentation website.

## What's Here

| File/Dir | Purpose |
|----------|---------|
| `plans.md` | Active task plan for the current feature branch |
| `status.md` | Session-to-session checkpoint log |
| `repo-operating-system.md` | Agent workflow rules and repository discipline |
| `prompts/` | Reusable AI agent starter prompts for ASCP workstreams |
| `superpowers/specs/` | Design specifications for completed features |
| `superpowers/plans/` | Implementation plans for completed features |

## Why Separate?

These files are for contributors and AI agents working on the repository. They contain task tracking, branch planning, and implementation details that would confuse or overwhelm users looking for protocol documentation.

For user-facing documentation, visit the [ASCP documentation website](/docs).
For developer-facing protocol reference, visit the [Reference section](/docs/reference).
