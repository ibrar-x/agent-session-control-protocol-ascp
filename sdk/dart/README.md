# Dart SDK

This directory is reserved for the ASCP Dart SDK package.

The Dart SDK is a later downstream target after the TypeScript SDK establishes the first package shape and integration approach.

The expected responsibilities come from `../../ASCP_Dart_SDK_Implementation_Plan.md`:

- typed ASCP models
- method wrappers
- event subscription support
- replay helpers
- transport abstraction
- validation hooks
- auth hooks
- Flutter-friendly async ergonomics

Do not start implementation here until the active plan explicitly moves to a Dart-focused feature branch.
